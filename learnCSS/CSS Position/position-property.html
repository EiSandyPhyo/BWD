Position: Static;

An element with postion: Static; is not positioned in any special way; it is always positioned according to the normal flow of the page:

This div element has position: static;

Position: Relative;

An element with position: Relative; is positioned relative to its normal position:

This div element has position: relative;

Position: Fixed;

An element with postion: Fixed; is positioned relative to the viewport, which means it always stays in the same palce even if the page is scrolled:

This div element has position: fixed;

Position: Absolute;

An element with postion: Absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).
However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.
Note: Absolute positioned elements are removed from the normal flow, and can overlap elements.
